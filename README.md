# Sistema de Gestão de RH

Este é um sistema de gestão de recursos humanos (RH) desenvolvido com backend em ASP.NET Core e frontend em React.

## Funcionalidades

- Gerenciamento de funcionários
- Controle de presença
- Cálculo de folha de pagamento
- Autenticação de usuários
- Dashboards para funcionários, gerentes e RH

## Tecnologias Utilizadas

### Backend
- ASP.NET Core
- Entity Framework Core
- SQL Server (ou outro banco de dados configurado)

### Frontend
- React
- Vite
- Tailwind CSS
- Axios para API

## Configuração e Execução

### Pré-requisitos
- .NET SDK (versão 8.0 ou superior)
- Node.js (versão 16 ou superior)
- Banco de dados (SQL Server recomendado)

### Backend
1. Navegue até a pasta `backend/`
2. Restaure as dependências: `dotnet restore`
3. Execute o projeto: `dotnet run`

### Frontend
1. Navegue até a pasta `frontend/`
2. Instale as dependências: `npm install`
3. Execute o projeto: `npm run dev`

## Estrutura do Projeto

- `backend/`: Código do servidor ASP.NET Core
- `frontend/`: Código do cliente React
- `dashboard fina.sln`: Arquivo de solução do Visual Studio
